Des. — Calculate Rules Config 改动点

  • Token 卡片动态增删：Separately 默认 1 个 Token，Combined 默认 2 个，均可通过 +/- 增删。与现有 Admin 逻辑一致。
  • Token Pair NEW：Anyone=任意交易对（现有）；Specific Pair=仅统计指定 Quote Token 交易对（如 USD1-MMT）。
  • Channel Multiplier ENHANCED：现有仅支持 Cex Limit 倍数。新增 Cex Instant 和 Dex 的独立倍数配置。
  • Daily Volume Boost NEW：勾选后展开每日倍数配置表。运营可为比赛期间每一天单独设置交易量倍数（如 Day1=3x, Day2=3x, Day3=2x...），未配置的天数默认 1x。支持动态增删行。
  • Chain ID：新增多链支持（ETH/Base/Solana/Sui），统一为「链名 (ID)」格式显示。

Des. — User Bonus Config 改动点

  • Enable 开关 NEW：勾选后展开配置项，不勾选则该活动无用户加成。
  • Eligible User Type：仅支持 Low Winner（低获奖次数用户） — 历史获奖次数 < N 的用户，需设置 Max Win Count 阈值。
  • Bonus Volume Multiplier：符合条件用户的交易量倍数（如 1.5x）。
  • 叠加计算公式：有效交易量 = 原始交易量 x 渠道倍数 x Daily Boost（如有）x 用户加成倍数（如有）
  • Max Bonus Cap：可选封顶金额，防止极端情况。留空则不设上限。
